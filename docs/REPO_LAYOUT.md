# Repository layout

Paths are relative to the project root.

## Top-level

| Path | Role |
|------|------|
| `pipeline/` | Library: `graph.py` (Neo4j + grounding), `planning.py`, `systems/` (S1–S5), `metrics/` (CCR, HTM, corpus scores). Import as `from pipeline…`. |
| `scripts/` | CLI entrypoints (see below). |
| `experiments/` | NER ablation, vector grounding helpers ([experiments/README.md](../experiments/README.md)). |
| `data/` | Segment JSONLs, templates ([data/README.md](../data/README.md)). |
| `results/` | **`ner_biollm/`**, **`ner_biollm_finetuned/`**, `cross_ner_comparison/`, `htm_vector_comparison/` (regenerated by `rerun_all.sh` unless skipped). **`ner_biollm_eval_snapshot_*`** folders hold frozen figures + `INTERPRETATION_OF_RESULTS_MT.md`. Translation JSONLs may be gitignored. |
| `archive/` | Retired segment JSONLs ([archive/data/section48/README.md](../archive/data/section48/README.md)); not used by `rerun_all.sh`. |
| `exports/` | Generated annotation CSVs ([exports/README.md](../exports/README.md)); not primary data. |
| `docs/` | Layout (this file), error schema, figures, notes ([docs/README.md](README.md)). |
| `tests/` | `pytest`. |
| `models/` | Checkpoints — not in git ([models/README.md](../models/README.md)). |
| `rerun_all.sh` | Batch driver: **BioMistral prompt + FT BioMistral NER**; eval, cross-NER dashboard, HTM threshold step. **Excludes segment `48_028`** (dense Tableau 2 block) unless `EXCLUDE_SEGMENT_IDS=` is empty. |

## Scripts

| Script | Role |
|--------|------|
| `run_pipeline.py` | Run translation systems → JSONL (`--exclude-segment-ids` optional). |
| `evaluate.py` | Corpus + CCR + optional **HTM** (Neo4j + `--gold-terms`); `--no-graph` skips CCR/HTM. |
| `plot_results.py` | Figures per condition. |
| `plot_cross_ner_dashboard.py` | Cross-condition comparison figures. |
| `prepare_data.py` | PDF → segments + NER JSONL. |
| `build_graph.py`, `extract_meddra.py` | Neo4j graph from MedDRA exports. |
| `compare_htm_vector_thresholds.py` | HTM threshold comparison plots (`ner_biollm` + `ner_biollm_finetuned` only). |
| `archive_results_snapshot.sh` | Optional `tar.gz` of `results/`, `exports/`, `planning_locks.json`. |
| `report_ambiguous_grounding.py` | CSV: ambiguous FR MedDRA keys across two segment JSONLs + planning lock overlap. |
| `sample_errors_for_annotation.py` | Sample rows + optional LLM pre-fill for manual error review. |

## Design

Importable code lives in **`pipeline/`**. **`scripts/`** stays flat. Supplementary training and audits live under **`experiments/`**.
